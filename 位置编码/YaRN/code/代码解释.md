# 代码解释：RoPE 与 YaRN

---

## 整体结构

```
precompute_freqs_cis()
    │
    ├── Step 1：计算基础频率 θ_i         ← 标准 RoPE
    ├── Step 2：YaRN 频率修正             ← 按圈数分段缩放
    └── Step 3：生成 cos/sin 缓存表

apply_rotary_pos_emb()
    └── 将 cos/sin 作用到 q, k 上
```

---

## Step 1：计算基础频率 θ_i

**原始代码（难读）：**
```python
freqs, attn_factor = 1.0 / (rope_base ** (torch.arange(0, dim, 2)[: (dim // 2)].float() / dim)), 1.0
```

**等价简化写法：**
```python
i = torch.arange(0, dim // 2, dtype=torch.float32)   # i = 0, 1, 2, ..., dim/2 - 1
freqs = 1.0 / (rope_base ** (2 * i / dim))            # θ_i = 1 / base^(2i/d)
attn_factor = 1.0
```

**对应公式：**

$$\theta_i = \frac{1}{\text{base}^{2i/d}}, \quad i = 0, 1, \dots, \frac{d}{2}-1$$

`freqs` 是一个长度为 `dim/2` 的向量，每个元素就是一个 $\theta_i$，频率随 $i$ 增大单调递减：

| 维度 $i$ | $\theta_i$（base=1e6） | 含义 |
|:---:|:---:|:---|
| 0 | 1.0 | 最高频，转圈最快 |
| ... | ... | ... |
| dim/2-1 | 接近 0 | 最低频，转圈最慢 |

## Step 2：YaRN 频率修正

只有当 **推理长度 > 训练长度** 时才触发。

### 2.1 解析配置参数

```python
orig_max    = rope_scaling.get("original_max_position_embeddings", 2048)  # L_train
factor      = rope_scaling.get("factor", 16)                               # s = L_test / L_train
beta_fast   = rope_scaling.get("beta_fast", 32.0)                          # 高频阈值 τ_high
beta_slow   = rope_scaling.get("beta_slow", 1.0)                           # 低频阈值 τ_low
attn_factor = rope_scaling.get("attention_factor", 1.0)                    # Attention 温度系数
```

### 2.2 找到需要过渡的维度边界

**原始代码：**
```python
inv_dim = lambda b: (dim * math.log(orig_max / (b * 2 * math.pi))) / (2 * math.log(rope_base))
low  = max(math.floor(inv_dim(beta_fast)), 0)
high = min(math.ceil(inv_dim(beta_slow)),  dim // 2 - 1)
```

**公式推导：**

训练时第 $i$ 个频率分量"转过的圈数"为：

$$r_i = \frac{\theta_i \cdot L_{train}}{2\pi} = \frac{L_{train}}{2\pi \cdot \text{base}^{2i/d}}$$

给定目标圈数 $\beta$，反解出对应的维度 $i$：

$$i = \frac{d \cdot \log\!\left(\dfrac{L_{train}}{\beta \cdot 2\pi}\right)}{2 \cdot \log(\text{base})}$$

这就是 `inv_dim(b)` 做的事。于是：

- `low`  = $i$ 使得 $r_i = \beta_{fast}$（圈数 = 32 对应的维度）→ **高于此维度的 $\theta_i$ 转圈够多，直接保留**
- `high` = $i$ 使得 $r_i = \beta_{slow}$（圈数 = 1 对应的维度）→ **低于此维度的 $\theta_i$ 转圈不足，需要压缩**

### 2.3 构建平滑过渡掩码 γ_i

**原始代码：**
```python
ramp = torch.clamp((torch.arange(dim // 2).float() - low) / max(high - low, 0.001), 0, 1)
```

**等价写法（更清晰）：**
```python
ramp = (torch.arange(dim // 2).float() - low) / (high - low)  # 线性插值
ramp = torch.clamp(ramp, min=0.0, max=1.0)                     # 限制在 [0, 1]
```

**对应公式（γ_i）：**

$$\gamma_i = \begin{cases} 0, & i < \text{low} \\ \dfrac{i - \text{low}}{\text{high} - \text{low}}, & \text{low} \le i \le \text{high} \\ 1, & i > \text{high} \end{cases}$$

直觉上：
- `ramp = 0` → 这个频率需要被完整压缩（低频，内插）
- `ramp = 1` → 这个频率完全保留（高频，外推）
- `0 < ramp < 1` → 线性混合过渡

### 2.4 混合得到新的 θ_i

**原始代码：**
```python
freqs = freqs * (1 - ramp + ramp / factor)
```

**公式等价展开：**

$$\theta_i^{new} = \theta_i \cdot \left[(1 - \gamma_i) \cdot \frac{1}{s} + \gamma_i \cdot 1\right] = \theta_i \cdot \left[\gamma_i + \frac{1-\gamma_i}{s}\right]$$

其中 $s = \frac{L_{test}}{L_{train}}$ = `factor`。

| $\gamma_i$ 的值 | 效果 |
|:---:|:---|
| 1 | $\theta_i^{new} = \theta_i$，完全保留，直接外推 |
| 0 | $\theta_i^{new} = \theta_i / s$，完整压缩，等价于位置内插 |
| 中间值 | 线性混合，平滑过渡 |

## Step 3：生成 cos/sin 缓存表

```python
t     = torch.arange(end)            # 位置编号 m = 0, 1, ..., end-1
freqs = torch.outer(t, freqs)        # freqs[m, i] = m * θ_i^new
```

`torch.outer` 计算外积，得到一个形状为 `[end, dim/2]` 的矩阵，每个元素就是：

$$\text{freqs}[m, i] = m \cdot \theta_i^{new}$$

然后将其复制拼接成完整维度，并乘以温度系数：

```python
freqs_cos = torch.cat([cos(freqs), cos(freqs)], dim=-1) * attn_factor
freqs_sin = torch.cat([sin(freqs), sin(freqs)], dim=-1) * attn_factor
```

形状从 `[end, dim/2]` 扩展为 `[end, dim]`，这里前后两半完全相同是为了和后续的 `rotate_half` 配合实现复数旋转（见下文）。

---

## apply_rotary_pos_emb：将位置编码作用到 q, k

### rotate_half 的几何含义

```python
def rotate_half(x):
    return torch.cat((-x[..., x.shape[-1] // 2:], x[..., : x.shape[-1] // 2]), dim=-1)
```

把向量的前后两半互换，并对后半取负：

$$[a_0, a_1, \dots, a_{d/2-1},\ b_0, b_1, \dots, b_{d/2-1}] \xrightarrow{\text{rotate\_half}} [-b_0, -b_1, \dots, -b_{d/2-1},\ a_0, a_1, \dots, a_{d/2-1}]$$

### 旋转公式

```python
q_embed = q * cos + rotate_half(q) * sin
k_embed = k * cos + rotate_half(k) * sin
```

**对应的复数旋转公式：**

将 $q$ 中每两个相邻维度视为一个复数 $q_j = a_j + ib_j$，RoPE 的旋转操作就是：

$$q_j' = q_j \cdot e^{im\theta_j} = (a_j + ib_j)(\cos m\theta_j + i\sin m\theta_j)$$

展开得到实部和虚部：

$$a_j' = a_j \cos m\theta_j - b_j \sin m\theta_j$$
$$b_j' = a_j \sin m\theta_j + b_j \cos m\theta_j$$

用矩阵表达：

$$\begin{pmatrix} a_j' \\ b_j' \end{pmatrix} = \begin{pmatrix} \cos m\theta_j & -\sin m\theta_j \\ \sin m\theta_j & \cos m\theta_j \end{pmatrix} \begin{pmatrix} a_j \\ b_j \end{pmatrix}$$

这正是 `q * cos + rotate_half(q) * sin` 所计算的内容（其中 `rotate_half` 完成了符号变换），只不过这里 `cos/sin` 缓存表的前后两半相同，实现了对所有维度对的并行计算。

## 总结：核心公式速查

| 步骤 | 公式 | 代码 |
|:---|:---|:---|
| 基础频率 | $\theta_i = \text{base}^{-2i/d}$ | `freqs` |
| 训练时圈数 | $r_i = \theta_i L_{train} / 2\pi$ | `inv_dim()` 反算 |
| 过渡边界 | $i_{low/high}$ 由圈数阈值反解 | `low, high` |
| 过渡系数 | $\gamma_i \in [0,1]$ 线性插值 | `ramp` |
| 新频率 | $\theta_i^{new} = \theta_i[\gamma_i + (1-\gamma_i)/s]$ | `freqs * (1 - ramp + ramp/factor)` |
| 位置相位 | $\phi_{m,i} = m \cdot \theta_i^{new}$ | `torch.outer(t, freqs)` |
| 旋转应用 | $q' = q\cos\phi + \text{rotate\_half}(q)\sin\phi$ | `apply_rotary_pos_emb` |
